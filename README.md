# AI-Enhanced-Documentation-Generator 📁
This program or tool i built will be able to help create, edit and format documents automatically for users.

## What I learnt 

## how can it help the community 
- Improves the accessibility to information.
- Reduces workload meaning less cognitive load making it easier from users to learn.
- reduces paperwork in a system.
- improves quality of the system / documentation.
- saves time and effort for user to get the workload done.
- Will be able to document what user did what work - so that the boss can keep track of progress.

## Features 
- creates content automatically for example displaying the template needed to format a letter or newspaper.
- customizes documentation
- automatic formating
- summarises and organises information
- Track work done
